# gophersay
Make a gopher say whatever you want.

## Examples
```
    `.-::::::-.`                            
.:-::::::::::::::-:. =======================
`_:::    ::    :::_` = hello gopher world! =
 .:( ^   :: ^   ):.  =======================
 `:::   (..)   :::.                         
 `:::::::UU:::::::`                           
 .::::::::::::::::.                         
 O::::::::::::::::O                         
 -::::::::::::::::-                         
 `::::::::::::::::`                         
  .::::::::::::::.                          
    oO:::::::Oo                             
```
